# 📰 Fake News Detection using Machine Learning

## 📌 Project Overview

Fake News Detection is an AI/ML-based project developed to identify whether a news article is **Real** or **Fake** using Natural Language Processing (NLP) and Machine Learning techniques.

The main objective of this project is to reduce the spread of misinformation by automatically analyzing news articles and predicting their authenticity.

---

# 🚀 Features

- Detects fake and real news articles
- NLP-based text preprocessing
- Machine Learning classification model
- Fast and accurate prediction system
- Real-time news classification
- User-friendly interface
- Model performance evaluation

# 📂 Dataset Information

This project uses the **Fake and Real News Dataset** to train and test the machine learning model for detecting fake news articles.

## Dataset Files

- `True.csv` → Contains genuine/real news articles
- `Fake.csv` → Contains fake and misleading news articles

## Dataset Columns

- `title`
- `text`
- `subject`
- `date`

---

# 📊 Dataset Source

The dataset used in this project is publicly available on Kaggle:
