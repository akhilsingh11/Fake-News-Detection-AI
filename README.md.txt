# Fake News Detection AI

AI-powered Fake News Detection web application using Flask, Machine Learning, and OpenRouter API.

## Features

- Detects fake and real news
- Logistic Regression
- Random Forest
- Gradient Boosting
- Decision Tree
- AI explanation system
- Confidence score

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Machine Learning

## Run Project

pip install -r requirements.txt

python app.py
